# Komentář k zákonu č. 326/1999 Sb., o pobytu cizinců na území České republiky (ZPC)

Veřejnoprávní úprava vstupu, pobytu a vycestování cizinců ze třetích zemí i občanů EU — přechodný a trvalý pobyt, víza, zaměstnanecká a modrá karta, správní vyhoštění a zajištění, zařízení pro zajištění cizinců, informační systémy a působnost cizinecké policie a Ministerstva vnitra. Transpozice řady směrnic EU (návratová 2008/115/ES, sloučení rodiny 2003/86/ES, rezidenti 2003/109/ES, studenti a výzkum 2016/801/EU, volný pohyb 2004/38/ES) a vazba na vízový kodex a Schengenský hraniční kodex.

Konsolidované znění k **1. lednu 2026** (dokument e-Sbírky `SB-1999-00326_2026-01-01`, document_id 2057043).
